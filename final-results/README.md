# Final Results Layout

This directory holds the reproduction scripts and human-study inputs for the evaluation. Generated outputs are **not** shipped with the artifact; they are produced when you run the scripts.

- `data/human-data` — shipped (human-study inputs)
- `data/description-data` — not shipped; you create it from your pipeline runs (see below)
- `scripts` — shipped (analysis scripts)
- `eval` (`eval/tables`, `eval/plots`) — generated by the scripts at run time

## Data

### `data/human-data`
Human-study inputs, included in the artifact:
- `Human Evaluation Survey- Pull Request Descriptions.csv`
- `survey_researcher_key.md`

### `data/description-data`
Judged-output JSONs consumed by the description-level analysis scripts. These are large and are **not** included in the artifact — generate them by running the pipeline (`description-generation/main.py` then `judge/judge.py`) and place the resulting judge files here. The scripts expect filenames of two shapes:
- `descriptions-<dataset>-modes-...-judge-<timestamp>.json`
- `results_<dataset>_<c|z>.json`

Note:
- Files containing `parsed` correspond to the curated dataset from Trudeau et al. and are reported as `Trudeau` in evaluation outputs.

## Scripts

All scripts write their tables to `eval/tables/` and plots to `eval/plots/`.

- `scripts/analyze_descriptions.py` — LLM-judge score summary per dataset and mode (reads `data/description-data`).
- `scripts/analyze_failure_reasons.py` — failure-category taxonomy (reads `data/description-data`).
- `scripts/analyze_lexical_metrics.py` — BLEU / ROUGE overlap and deltas from raw (reads `data/description-data`).
- `scripts/analyze_human_evaluation.py` — human rankings/preferences (reads `data/human-data`).
- `scripts/analyze_human_llm_agreement.py` — human vs. LLM-as-judge agreement (reads both).

## Run

```bash
cd final-results

# Runs against the included human-study inputs:
python3 scripts/analyze_human_evaluation.py

# Require data/description-data/ populated from your pipeline runs:
python3 scripts/analyze_descriptions.py
python3 scripts/analyze_failure_reasons.py
python3 scripts/analyze_lexical_metrics.py
python3 scripts/analyze_human_llm_agreement.py
```
